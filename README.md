Self Study Object Orientation Programming
=========================================
### Initial Setup ###
1. Make a new virtualenv: ``python3.5 -m venv .venv``
2. Activate the virtualenv: ``source env/bin/activate``
3. Install: ``pip install -r requirements.txt``
